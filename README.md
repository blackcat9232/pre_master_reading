# Pre-Master Reading Notes

Personal literature reading notes in optics and photonics, maintained as an [Obsidian](https://obsidian.md) vault with Git version control.

## Topics

- **Electromagnetic metasurfaces** — physics, design principles, and applications
- **Gradient metasurfaces** — phase-gradient mechanisms, wavefront shaping, flat optics
- **Metasurface reviews** — comprehensive surveys covering homogeneous/inhomogeneous metasurfaces, dielectric metasurfaces, active/tunable devices, and nonlinear metasurfaces

## Repository Structure

Each subfolder corresponds to one paper and follows a consistent layout:

```
paper_name/
├── paper_name.md                  # Reading notes (Obsidian markdown)
├── paper_name_origin.pdf          # Original paper PDF
├── paper_name_layout.pdf          # Annotated/layout PDF
├── paper_name_content_list.json   # Extracted content index
├── paper_name_content_list_v2.json
├── paper_name_middle.json         # Intermediate processing metadata
├── paper_name_model.json          # AI model interaction metadata
└── images/                        # Figures and extracted images
```

## Tooling

- **Editor:** [Obsidian](https://obsidian.md) with the [RealClaudian](https://github.com/bluebrother/realclaudian) plugin
- **AI-assisted reading:** Notes are generated with the help of AI tools to extract paper structure, summarize key points, and organize figures
- **Version control:** Git for tracking note revisions
